# Eco-Friendly Furniture Store Website

A fully responsive multi-page website showcasing sustainable furniture products and eco-friendly initiatives.

## Features
- Home page with hero section, product highlights, showroom locations, facilities, and testimonials
- About Us page with brand mission and sustainability practices
- Products page with eco-friendly furniture listings
- Blog page with latest posts and comments section
- Contact page with embedded Google Maps and functional contact form
- Responsive design for mobile and desktop
- Reusable components for consistent UI
- Modern typography using Google Fonts and icons with Font Awesome

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Interactive navigation)
- Google Fonts & Font Awesome

## Key Learnings
- Frontend development skills for multi-page responsive websites
- UI/UX design principles
- Structuring content for readability and engagement
- Interactive elements and form handling
